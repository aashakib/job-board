#Job Board

## Installation

- Clone the project first
- Run these commands  
    *`cp .env.example .env` to create env file*  
    *`composer install`*  
    *`php artisan key:generate`*
    *`php artisan migrate --seed`*

- Set Write permission to  
    *`SUDO CHMOD -R 777 bootsrap/cache`*  
    *`SUDO CHMOD -R 777 storage`*
